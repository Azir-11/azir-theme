# Azir Theme for Zed

This directory contains Azir theme files for the Zed editor.

## Installation

1. **Download the theme files**: Copy the theme files from this directory to your Zed themes folder:
   - **macOS/Linux**: `~/.config/zed/themes/`
   - **Windows**: `%APPDATA%\Zed\themes\`

   You can copy all individual files or just the combined `azir.json` file.

2. **Restart Zed** (if it's running)

3. **Select the theme**:
   - Open the command palette: `Cmd/Ctrl + Shift + P`
   - Type "theme selector" and press Enter
   - Choose your preferred Azir theme variant

## Available Themes

### Individual Theme Files
- `light.json` - Azir Light Default
- `light-high-contrast.json` - Azir Light High Contrast
- `dark.json` - Azir Dark Default (recommended for dark mode users)
- `dark-high-contrast.json` - Azir Dark High Contrast
- `dark-dimmed.json` - Azir Dark Dimmed (softer dark theme)

### Combined Theme File
- `azir.json` - Contains all 5 theme variants in a single file

## Usage

If you install the combined `azir.json` file, you'll have access to all theme variants from Zed's theme selector. If you prefer, you can install only the individual theme files you want to use.

## Theme Variants

- **Light Default**: Clean, bright theme matching GitHub's light mode
- **Light High Contrast**: Higher contrast version for better accessibility
- **Dark Default**: Dark theme matching GitHub's dark mode
- **Dark High Contrast**: Higher contrast dark theme for better accessibility
- **Dark Dimmed**: Softer, less intense dark theme for extended coding sessions

## Customization

To customize the theme, you can:
1. Copy the theme file you want to customize
2. Rename it (e.g., `azir-custom.json`)
3. Modify the colors in the `style` section
4. Save and select your custom theme in Zed
